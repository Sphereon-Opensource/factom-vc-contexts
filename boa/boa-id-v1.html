Terminology

issueDate

Date at which the BOA ID was issued.

validityDuration

The time frame in during which the BOA ID is valid (measured as started from the issue date).

name

The full name of the BOA to which this ID has been issued.

boaNumber

The identifying BOA number.

idPhoto

A base 64 ID image of the BOA to which the BOA ID has been issued.

domain

The domain over overseen by the specified BOA. Currently there are 6 possible domains:
  1. Openbare ruimte (public spaces)
  2. Milieu, welzijn en infrastructuur (environment, wellness, and infrastructure)
  3. Onderwijs (education)
  4. Openbaar vervoer (public transport)
  5. Werk, inkomen en zorg (work, income, and care)
  6. Generieke opsporing (generic detection)
