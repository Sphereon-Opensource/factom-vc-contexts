Terminology

validFrom

Date from which the BOA ID is valid.

validUntil

The date after which the BOA ID is no longer valid.

name

The given name name of the BOA to which this ID has been issued.

name

The family name of the BOA to which this ID has been issued.

name

Any additional name of the BOA to which this ID has been issued (usually used for middle names).

boaNumber

The identifying BOA number.

idPhoto

A base 64 ID image of the BOA to which the BOA ID has been issued.

domain

The domain over overseen by the specified BOA. Currently there are 6 possible domains:
  1. Openbare ruimte (public spaces)
  2. Milieu, welzijn en infrastructuur (environment, wellness, and infrastructure)
  3. Onderwijs (education)
  4. Openbaar vervoer (public transport)
  5. Werk, inkomen en zorg (work, income, and care)
  6. Generieke opsporing (generic detection)

employer

The organization that employs the specified BOA.
locality
The administrative region in which the BOA is active.
