Terminology

validFrom

The date and time from which the ID is valid.

validUntil

The date and time after which the ID is no longer valid.

givenName

The given name name of the BOA to which this ID has been issued.

familyName

The family name of the BOA to which this ID has been issued.

additionalName

The additional name(s) of the BOA to which this ID has been issued, if any (usually used for middle names).

idNumber

The identifying BOA number as assigned by the employer.

idImage

The base64url encoded (RFC 4648) image of the person to which the BOA ID has been issued.

domain

The domain overseen by the specified BOA. Currently there are 6 possible domains:
  1. Openbare ruimte (public spaces)
  2. Milieu, welzijn en infrastructuur (environment, wellness, and infrastructure)
  3. Onderwijs (education)
  4. Openbaar vervoer (public transport)
  5. Werk, inkomen en zorg (work, income, and care)
  6. Generieke opsporing (generic detection)

employer

The organization that employs the specified BOA.
locality
The administrative region in which the BOA is active.
