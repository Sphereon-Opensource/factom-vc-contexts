CAdESRSASignature2020

This class represents a linked data signature suite. See Linked Data Signature.

Status
unstable
Expected properties
type, created, verificationMethod, proofPurpose, cades

Properties

cades
The CMS signature (enveloping) in PKCS7 format of the LD document.

Example

"proof": {
    "type": "CAdESRSASignature2020",
    "created": "2020-12-15T12:26:23Z",
    "verificationMethod": "did:example:123#eidas-key",
    "proofPurpose": "assertionMethod",
    "cades": "-----BEGIN PKCS7-----\r\nMIIERgYJKoZIhvcNAQcC...i5i+4LlTEA\r\n-----END PKCS7-----\r\n"
}
